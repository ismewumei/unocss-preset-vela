# @mi/unocss-preset-vela

The vela preset for [UnoCSS](https://github.com/unocss/unocss).

## Installation

```bash
npm i -D @mi/unocss-preset-vela
```

```ts
import presetVela from '@mi/unocss-preset-vela'

Unocss({
  presets: [
    presetVela(),
  ],
})
```
